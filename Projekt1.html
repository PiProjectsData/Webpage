Die Software

Für dieses Projekt sollte vorhanden sein:

Code für LED Matrix 4x8 - "heart.py"

Das ist der Code mit dem wir anfangen werden.
#!/usr/bin/env python
import time
import unicornhat as unicorn

print("""heart.py Der Pi leuchtet jetzt.""")

unicorn.set_layout(unicorn.AUTO)
unicorn.rotation(0)
unicorn.brightness(0.5)
width,height=unicorn.get_shape()
#Herz blau
unicorn.set_pixel(0,0,0,0,0)
unicorn.set_pixel(0,1,0,0,0)
unicorn.set_pixel(0,2,0,0,255)
unicorn.set_pixel(0,3,0,0,0)
unicorn.set_pixel(1,0,0,0,0)
unicorn.set_pixel(1,1,0,0,255)
unicorn.set_pixel(1,2,0,0,255)
unicorn.set_pixel(1,3,0,0,255)
unicorn.set_pixel(2,0,0,0,255)
unicorn.set_pixel(2,1,0,0,255)
unicorn.set_pixel(2,2,0,0,255)
unicorn.set_pixel(2,3,0,0,0)
unicorn.set_pixel(3,0,0,0,0)
unicorn.set_pixel(3,1,0,0,255)
unicorn.set_pixel(3,2,0,0,255)
unicorn.set_pixel(3,3,0,0,255)
unicorn.set_pixel(4,0,0,0,0)
unicorn.set_pixel(4,1,0,0,0)
unicorn.set_pixel(4,2,0,0,255)
unicorn.set_pixel(4,3,0,0,0)
unicorn.show()
time.sleep(2)
Nachdem du den Code gespeichert hast musst du zum Ausführen den Terminal öffnen und mit ls und cd zum Desktop, bzw zu dem Ort, wo die Datei liegt, gehen. Gebe dann ein:
sudo Python heart.py
Dann sollte das im Terminal ausgegeben werden: "heart.py Der Pi leuchtet jetzt." und auf der LED Matrix ein blaues Herz aufleuchten.

Nachdem du dein Herz mit Freude begutachtet hast gehen wir den Code mal durch. Die ersten 3 Zeilen ignorieren wir einfach mal. Das print ist dafür da, dass wir wenn wir das Programm ausführen auf dem Terminal etwas ausgeben können. Hier sehen wir ob es läuft und mit welchem Programm. Dieses hier heißt "heart.py". Es ist einfach eine Bestätigungsnachricht für uns, die man jederzeit ändern kann. Du kannst innerhalb der "" deinen Namen schreiben zB:
print("Ich bin Lena.\n Und habe mit Python ein Herz auf dem Raspberry Pi programmiert.")
Tipp: Wenn du das Programm änderst und kannst du zum Speichern Strg und S drücken.
Im Terminal kannst du einfach mit den Pfeiltasten nach oben und unten duch deine letzten Befehle durchgehen. Dann musst du nicht jeden Befehl nochmal eintippen.

Wichtig für uns ist erst ab Zeile 10. Das unicorn.set_pixel(0,0,0,0,0) setzt den Pixel (0,0) auf der Matrix (Koordinatensystem) auf den Wert (0,0,0). Das heißt unser erster Pixel, der bei x = 0 und y = 0 liegt, ist komplett aus, weil wir ihm keine Lichtwerte geben. Die drei Zahlen stehen für R = Rot, G = Grün, B = Blau. Und die Zahlen können von 0 bis 255 gehen. Das heißt von ganz aus bis volle Intensität. Zum Vergleich was wir zwei Zeilen drunter sehen, unicorn.set_pixel(2,0,0,0,255) färbt den Pixel an Stelle x=2, y=0 zu r=0,g=0,b=255 --> also Blau. Zb ist dann 255,0,255 Lila, weil wir die volle Intensität von Rot und Blau nehmen. Wenn sie beide gleichzeitig leuchten, nimmt unser Auge eine Mischfarbe wahr, nämlich Lila.

Aufgabe für dich:

Welcher RGB-Wert wäre dann die Farbe Grün oder Gelb?
Welche Farbe ist bei 255,69,0?
Probiere mal für die Farbwerte verschiedene Zahlen aus. Dazu kannst du auch auf dieser Webseite sehen wie man die Farben mischt: RGB Color Codes Chart oder hier Farbtabelle.
Das time.sleep(2) ist zuständig, wie lange das Herz angezeigt wird. Wenn du also mal statt 2 eine 5 oder 0.3 machst, wirst du einen Unterschied sehen. Welchen?
Und damit man überhaupt etwas sehen kann ist der unicorn.show() Befehl zuständig. Das heißt, wenn du den um paar Zeilen weiter nach oben verschiebst, wird man das was danach kommt nicht sehen, weil er nur das zeigt, was zu diesem Zeitpunkt gesetzt ist.
Soooo deine nächste Aufgabe ist zu überlegen was passiert, wenn wir die Zeilen, an der für RGB nur 0 steht, weg machen. Wird das Herz dann immer noch richtig ausgegeben?
Ja und somit sparen wir auch Platz bzw Code. Das heißt Die LEDS sind defaultmäßig / standartmäßig aus.
Tu mal den Code vom ersten unicorn.setPixel bis time.sleep kopieren und setze dort eine andere Farbe rein. Was passiert?
Das Herz wird zwei mal in verschiedenen Farben hintereinander leuchten. Das heißt unser Programm arbeitet sich von oben nach unten durch.

Jetzt wissen wir so ziemlich viel um etwas auf der Led Matrix auszugeben. Jetzt bist du dran! Nehm am besten ein kariertes Blatt vor dich und zeichne die Größe der Matrix nach. (4x8 oder 8x8 Kästchen) Jetzt kannst du darin ein Bild mit sogar alles Farben sozusagen malen und dann auf deine LED Matrix projezieren. Dazu einfach Pixel für Pixel durchgegen und die Farbe setzen. Zum Beispiel: Zahlen, Buchstaben, Smileys, Katzenkopf, Regenbogen, Pixelart.
Viel Erfolg! :D

Programmierkenntnisse Erweitern

Hast du dich gut in den Quellcode eingelebt? Dann können wir den nächstesn Schritt wagen und den Code vereinfach, was aber nicht automatisch einfach zu verstehen ist. Stellt euch vor ihr habt einen Stuhl. Auf diesem Stuhl kann immer jemand draufsitzen. Aber ihr wisst nicht wer. Aber ihr wisst, dass da immer jemand beim Essen draufsitzt. Das heißt wir programmieren sozusagen diesen Stuhl. Das wäre dann eine Variable. Ähnlich wie im Matheunterricht das x. Also nehmen wir zb wenn wir das blaue Herz nehmen alle 255 raus und schreiben "wert" hin. Oben schreiben wir dann wert = 255 hin. Somit wird sich das Programm jedes mal wenn er wert sieht schauen welchen Wert das hat. Wenn ihr es richtig gemacht habt sollte es wie davor funktioniert haben.
Mit dem Schritt haben wir eine Variable erstellt und ihr den Namen wert und den Wert 255 gegeben. Das wäre also vergleichbar mit: Wir haben einen Stuhl erstellt und haben Papa draufgesetzt. Aber jetzt kann nur Papa draufsitzen, weil er nicht mehr gebeten wird aufzustehen. Wenn du wieder das Herz kopierst und unter dem anderen einfügst, kannst du vor dem zweiten sagen, dass Mama drauf sitzen soll. Das heißt wir machen wert = 50.
Jetzt haben wir das Verständnis für eine Variable erlernt. Ganz cool wird das dann, wenn wir 3 Variablen erstellen. Rot, Grün und Blau. Achte darauf, dass Variablennamen immer klein geschrieben sind. Besser wäre sogar, wenn wir r für Rot, g für Grün und b für Blau nehmen. also ersetzten wir alle 3.4. und 5. Zahl mir r, g, b. Zum Beispiel = unicorn.set_pixel(3,2, r,g,b) Jetzt müssen wir die Variablen mit einem Wert definieren. Zum Beispiel
r = 255
g = 212
b = 0
Achte darauf, dass das vor dem Benutzen steht. Sonst weiß das Programm zu diesem Zeitpunkt nicht, welchen Wert die Variable hat und gibt einen Fehler aus.

Wenn das gut geklappt hat können wir eine Schleife erstellen.
Ihr merkt bestimmt, dass die ersten zwei Zahlen (x und y) sich nicht viel unterscheiden. Und zwar geht es einmal von 0 bis 3 und von 0 bis 7. Nicht mehr und auch nicht weniger. Das ist die Größe der Matrix Das heißt wir machen einmal die Variable x und die Variable y. indem wir diesen Code einfügen:
for x in range (0-3):
  for y in range (0-7):
    unicorn.set_pixel(x,y,r,g,b)
Sehr sehr wichtig ist, dass es richtig eingerückt ist.
Um deine Python Kenntnisse zu erweitern kannst du if, else und arrays lernen. "If - else" heißt soviel wie "falls - ansonsten". Mit einem array können wir das Herz etwas anders definieren. Auch sind Objekte und Gui möglich um während der Laufzeit zu entscheiden, was auf der Matrix ausgegeben werden soll. Natürlich kann die LED Matrix noch mehr: Mehr über unicorn lernen.

Sonstige Erweiterungen

Man kann zusätzlich (für 8x8 Matrix) einen Diffuser drucken lassen. Das ist ein Gitter um die Pixel zu rastern. Dazu ist ein 3D-Drucker nötig. Siehe hier die Durckvorlage Dann muss noch eine Plexiglasscheibe zur Vollendung draufgeklebt werden. Somit wird ein Pixeliger Effekt erstellt, den ihr von Tetris bestimmt kennt.
